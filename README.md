# BMI-IMC Calculator

Calculateur d'Indice de Masse Corporelle (IMC) gratuit, multilingue et responsive.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation

```sh
npm install
npm run dev
```
